Como criar uma Branch
Caso queremos modificar algo no projeto e não alterar o que ja foi feito na Main. 

Para fazer isso usamos: git checkout -b "nome-branch" a branch não precisa ser necessáriamente um novo arq. mas pode ser alteração em arq existente. E lembre que o espaço no nome é com -

Para retornar para a MAIN é só refazer oo processo mudando o nome. Mas sem a ncessidade do -b 

Para juntar a Branch com a Main usamos o comando Git Merge + nome da Branch, mas temos que estar na main.
